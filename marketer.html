Hey — I'm Marvin Nyalia Ochieng, a fullstack developer and cybersecurity specialist based in Nairobi, Kenya. I've spent years learning how digital systems are built, and more importantly, how they break — so I can make them bulletproof.

My journey started with a curiosity about how websites worked, which quickly grew into a deep interest in both building robust web applications and understanding the security vulnerabilities that live inside them. That dual perspective is what makes me different — I don't just write code, I think like an attacker while building like an engineer.

On the development side, I'm comfortable across the entire stack — from crafting pixel-perfect frontends with React to architecting APIs with Node.js and Python.
